Display Property

The display property allows you to control how an element is displayed on a web page.

Block-Level Elements

Block-level elements take up as much horizontal space as possible by default.

Examples of block-level elements are <p>, <ol>, <dl>, <ul>, and all heading tags.

Inline-Level Elements

Inline elements display within a line and do not force the text after them to start on a new line.

Examples of inline elements are <a>, <em>, <span>, and <b>.

Property for Display (Keywords)

inline – Displays elements in a horizontal layout.

block – Displays elements in a vertical layout.

list-item – Displays elements as list items.

none – Hides the element; it does not affect the layout.

inherit – Inherits the display value from the parent element.

Reflection
