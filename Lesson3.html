Display Property

Allows you to control how an element is displayed


Block-Level Elements

Block level elements take up as much horizontal space as possible by default.

Examples of this are the <p> , <ol> , <dl> , <ul> , and all headings.

Inline Level Elements

Inline elements display in a line

They do not force the text after them to a new line.

Examples of this are <a> , <em> , <span> , <b> , and an anchor.

Property For Display (Keywords)

inline - Inline display or horizontal layout

block - Block display or vertical layout.

list-item - Display as a list item.

none - No box or the element has no effect onthe layout.

inherit - Inherit value from the parent.
