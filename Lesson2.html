Structure of CSS

Selectors, the HTML element you want to style

Property, the style attribute you want to modify

Value, he weight of the attribute you want to apply to the element property

Declaration, is a single instruction that defines a specific style for an element

Declaration Block, fundamental part of a CSS rule. It is a block of code that contains one or more CSS declarations


Stylesheet and its parts


A set of instructions to a Web browser on how to display various elements on a Web page is known as style sheet

Every CSS (whether it is contained in a .css file, or embedded in the head element of an HTML document) is a series of instructions called statement

stylesheet help separate content from appearance

stylesheet suggests the browser to display a page in a particular wa and how the pages should be displayed in the browser.

2 things to remember when using CSS

  • The type attribute should always be enclosed in the style tag with the value of the type attribute being "text/css"
  • CSS code should be enclosed in HTML comments <!--...--> so that in a web browser that does not understand CSS, the browser will not see the CSS code. But today, however, all current browsers support CSS

Three kinds of CSS

  1. External CSS
  2. Internal CSS
  3. Inline CSS

External CSS

  • easily allow you to change the layout of your entire web site by simply changing the external style sheet and every page is instantly update
  • has a .css file extension
  • the most global of the three kinds of CSS because you can apply the same one to an unlimited number of pages

Internal CSS

  • used for creating a document-wide style rule
  • placed between the head tag
  • embedded style sheets override external style sheets
  • Is placed in the container tag of style

Inline CSS

  • stored directly in the style attributes of the HTML tags
  • inline style sheets override external and embedded style sheets
  • uses the attribute style inside of the body tag
