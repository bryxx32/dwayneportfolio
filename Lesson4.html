Classes

What are Classes

  • is a user-defined selector.
  • used to apply a style rule to specific HTML tags.
  • is a reusable style definition that allows you to apply the same set of formatting rules to multiple HTML elements.
  • can control individual HTML element formatting
  • Are defined by Dots

Two Methods of writing Class

  1. indicating the HTML element followed by a dot then followed by the class name (any name you want for it)
  2. writing the class name preceded by a dot (.)

Class Selectors

Class selector (.classname), reusable across different elements.

Element + Class selector (p.classname),precise control over styling only one type of element.
