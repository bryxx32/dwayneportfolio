Setting Dimensions

CSS can be used to manage dimensions such as visibility, width or height of HTML elements.

Properties of Dimensions

Visibility - Indicates if an element is visible or hidden


Values

  • Visible, default value, the value is visible
  • Hidden, The element is hidden (but still takes up space)
  • Collapse, This value removes a row or column as if display: none where used. collapse is used on other elements, it renders as "hidden"

Width - Indicates the width


height - Indicates the height


Line-Height - Indicates the Line Spacing


Max-Height - Indicates the maximum height


Min-Height - Indicates the minimum height


Max-Width - Indicates the maximum width


Min-Width - Indicates the minimum width

Pseudo-Classes and Link

  • Pseudo-classes are defined by a colon and are not user-defined. Specific pseudo-classes can be usedon specific HTML elements only
  • The anchor element has four pseudo-classes: link,visited, hover and active. Each one affects the behavior of the anchor element.
  • LVHA or Link, Visited, Hover,Active is the order of the pseudo-classes.
  • The value that you place inside {â€¦} will be displayed whenever a pseudo-class property is performed. You can set the color of the text orelement, its background, its decoration and many other characters.

Properties and Their Description

:link - Sets unvisited link characteristic.

:visited - Sets visited link characteristic or effect.

:active - Sets the character of an active link element as when the user clicks the mouse on the link.

:hover - Sets the character of a link element as when the user hovers or moves the mouse over the link.
